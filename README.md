## 项目简介（中文 | [English](README_en.md)）

本项目旨在帮助开发者快速搭建知识文档，构建可部署的在线知识文档。

**主要内容包括：**

1. **项目结构介绍**
2. **文档语法撰写**

## 项目意义

帮助开发者快速搭建知识文档，构建可部署的在线知识文档。

## 目录结构说明

```bash
├─ code           <--- 代码目录
├─ data           <--- 数据目录
├─ docs           <--- 笔记目录
│  ├─ chapter1
│  │  ├─ images   <--- 静态资源
│  │  └─ xxx.md   <--- markdown笔记
│  ├─ chapterX
│  ├─ en          <--- 英文笔记目录
│  │  ├─ chapter1
│  │  │  ├─ images   <--- 静态资源
│  │  │  └─ xxx.md   <--- markdown笔记
│  │  └─ chapterX
│  │─ .nojekyll      <--- 防止Github忽视下划线开头文件
│  │─ _sidebar.md    <--- 目录页
│  │─ logo.svg
│  │─ index.html
│  └─ README.md      <--- 首页
├─ .gitignore
├─ README_en.md
└─ README.md
```
